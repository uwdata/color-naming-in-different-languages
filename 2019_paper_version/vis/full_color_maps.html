Color Maps

These visualizations represent how Korean and English divide up the whole color spectrum.

Each dot represents a 10x10x10 LAB color space. Each dot is given the average color of the most common name used for that color. The size of the dots is proportional to either what percent of names were that most common name (Max P(T|C)) or by the saliency measure of color name agreement.
